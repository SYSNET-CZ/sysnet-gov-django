# Changelog

## [0.1.0] - 2024-05-22
### Added
- Initial release of SYSNET Gov UI Boilerplate.
- Core JVS design tokens (colors, typography).
- Base components: Button, Input, Select, Icon, Logo.
- Layout components: Header, Sidebar, Card.
- Data components: Table, Alert.
- Storybook 8 configuration and base stories.
- Dark Mode foundations in CSS.
- Accessibility (A11y) features using Vue useId and ARIA.
