# SYSNET Gov UI

Moderní, výkonná a udržitelná knihovna Vue 3 komponent pro aplikace české státní správy.

## Proč SYSNET Gov UI?
- **Lehká váha**: Bez runtime režie Stencil.js (náhrada za GDS).
- **Tailwind Native**: Postaveno na Tailwind CSS v4.
- **Flexibilita**: Žádný Shadow DOM, plně přizpůsobitelné.
- **ISO/GDS kompatibilita**: Barvy a typografie striktně následují vládní manuály.

## Instalace
```bash
npm install @sysnet-cz/gov-ui
```

## Použití
```ts
import { GovButton } from '@sysnet-cz/gov-ui';
import '@sysnet-cz/gov-ui/style.css';
```
